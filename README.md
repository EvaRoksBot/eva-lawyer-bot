# Eva Lawyer Bot

Telegram bot that assists with legal document analysis and integrates with external services.

## Requirements
- Node.js 16+
- npm 10+

## Setup
1. Copy `.env.example` to `.env` and fill in the variables.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run tests:
   ```bash
   npm test
   ```
